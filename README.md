# login-page-html
# Mini Login System (Browser Only)

A simple **login/signup system** built entirely in **HTML, CSS, and JavaScript**.  
Runs **completely in the browser** using `localStorage` and `sessionStorage`.  
No backend, no Node.js, no installation required — perfect for **portfolio demos**.

---

## 🔹 Features

- Signup / Create Account
- Login with Email & Password
- Remember Me (localStorage)
- Session Login (sessionStorage)
- Dashboard page after login
- Logout
- Multiple users stored in browser
- Fully responsive and clean UI

---

## 🔹 Project Structure
login-demo/
│
├── index.html ← Login page
├── signup.html ← Signup page
├── dashboard.html ← Dashboard page
├── style.css ← CSS styling
└── app.js ← JavaScript logic

---

## 🔹 How It Works

1. User signs up → account saved in `localStorage`.
2. User logs in → authentication checked in the browser.
3. "Remember Me" → stores session in `localStorage`; otherwise `sessionStorage`.
4. Dashboard displays logged-in user.
5. Logout clears session data.

> ⚠️ This is for demo purposes only — **not secure for production**.

---

## 🔹 How to Use

1. Open `index.html` in your browser.
2. Signup a new account via `signup.html`.
3. Login using the email & password.
4. Check "Remember Me" to stay logged in across sessions.
5. Access the dashboard page and logout when done.
