# login-page-html
# Mini Login System (Browser Only)

A simple **login/signup system** built entirely in **HTML, CSS, and JavaScript**.  
Runs **completely in the browser** using `localStorage` and `sessionStorage`.  
No backend, no Node.js, no installation required — perfect for **portfolio demos**.

---

## 🔹 Features

- Signup / Create Account
- Login with Email & Password
- Remember Me (localStorage)
- Session Login (sessionStorage)
- Dashboard page after login
- Logout
- Multiple users stored in browser
- Fully responsive and clean UI
